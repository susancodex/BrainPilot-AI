Before generating any UI:

1. Analyze all backend modules.
2. Analyze all API endpoints.
3. Analyze all serializers/schemas.
4. Analyze all permissions.
5. Analyze all user roles.
6. Analyze all business logic.
7. Analyze all AI modules.
8. Analyze all background jobs.
9. Analyze all notification systems.
10. Analyze all analytics systems.

The frontend MUST expose EVERY backend capability.
